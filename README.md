# misskey-booru-rs
Bot for Misskey that posts an image from Gelbooru every n seconds. Supports running in Docker.

## Running
1. Clone the repository
1. Copy `config_sample.toml` to `config.toml`
1. Edit `config.toml`
1. Run `cargo run --release` or `docker-compose up`
